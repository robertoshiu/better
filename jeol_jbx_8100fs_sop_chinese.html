第三章：机器操作

3.1 启动程序

步骤 1：
在 BRK Lithography Core iLab Kiosk 中启用 JBX-8100FS
步骤 2：
打开 jbxwriter 软件。检查是否已最小化或在其他桌面标签页中
步骤 3：
检查载物台上是否已有晶圆盒，或装载锁中是否有晶圆盒
步骤 4：
检查当前设置的条件文件；必要时更改（确保 Restore 和 DEMAG 已勾选）
重要提醒： 切勿尝试打开新的 jbxwriter 实例，这可能导致工具通信问题。

3.2 样品安装

3.2.1 晶圆盒使用注意事项

小心操作！ 晶圆盒包含许多精密部件，本身也很敏感。它们是精密加工的，每个成本高达 30,000 美元。请轻柔操作，避免碰撞或掉落。

3.2.2 晶圆安装步骤

1.
将晶圆盒倒置放在无尘布上
2.
取下背面支架，将其正面朝上放置以确保接地针脚不受损坏
3.
对于 4" 和 6" 晶圆盒：首先将支架顶部的锁逆时针旋转约 120°
4.
用两根手指按住 V 型弹簧的边缘并旋转，直到脱离晶圆盒的悬垂部分
5.
小心地提起背面支架并将其正面朝上放在无尘布上

3.2.3 碎片安装

注意： 使用碎片比使用晶圆更复杂。如果可能，建议使用晶圆以节省时间和成本。碎片尺寸不应小于 10mm x 10mm。

3.3 装载晶圆盒

1.
打开交换室门
2.
确保"载物台上的晶圆盒"LED灯熄灭
3.
如有需要，按住 EVAC/VENT 按钮对装载锁进行通风
4.
打开装载锁，将晶圆盒放入袋中，插入装载锁
5.
按住 LOAD/UNLOAD 按钮直到开始闪烁

3.4 检查样品高度

1.
在 jbxwriter 中设置晶圆盒类型
2.
移动到样品中心或 P 标记位置
3.
使用 HEI 按钮检查样品高度
重要： 高度必须在 ±100μm 范围内，理想情况下在 ±50μm 范围内。接近 0μm 将获得更好的写入质量。

3.5 材料校正

3.5.1 非对齐曝光材料校正

1.
在校准页面的材料校正标签页中，点击加载按钮
2.
加载 0_currentnA.mcorrprm 文件
3.
在束流漂移子标签页中，点击"设置 BE 标记"，然后执行

3.5.2 对齐曝光材料校正

1.
使用 SEM 找到 P 标记
2.
在用户定义位置注册 P 标记
3.
加载或创建 MCORRPRM 文件
4.
确定 P 标记偏移量并更新全局标记检测偏移量

3.6 运行 DAILYCAL

建议： 每次更换电流后 15-40 分钟运行 DAILYCAL 批量校准文件。这有助于确保获得良好的结果。
DAILYCAL 包含：
  • 束流电流测量
  • AE 标记检测（2次）
  • 静态聚焦校正
  • BE 标记检测（2次）
  • 主偏转器校正
  • 子偏转器校正
  • 失真校正
  • 保存

3.7 VisualJob 和 Magazine 文件

1.
在 Beamer PC 上创建 V30、JDF 和 SDF 文件
2.
使用 U 盘将文件传输到 Linux PC
3.
更新 SDF 文件中的偏移量
4.
使用 SCHD GUI 或终端命令创建 MGN 文件

3.8 曝光

1.
在 jbxwriter 曝光页面中，移除队列中的所有杂志文件
2.
打开您的 MGN 文件
3.
开始曝光

3.9 卸载晶圆盒

1.
确保晶圆盒在原点位置
2.
打开交换室门
3.
按住 LOAD/UNLOAD 按钮卸载晶圆盒

3.10 完成工作

1.
为下一位用户加载电流（如需要）
2.
从晶圆盒上取下样品
3.
将晶圆盒放回塑料盒中
4.
关闭 Beamer 并注销 Beamer PC
5.
在 iLab Kiosk 中禁用 8100

4.4 标记组成示例

  • 永不放入 ICP RIE 腔室的样品：50nm 金 (Au) 在 5nm 钛/铬 (Ti/Cr) 上（用于粘附）

蚀刻工艺：
  • 深度：1μm（可以更薄，需要实验确定）
  • 宽度：必须 < 5μm
  • 深度范围：200-1000nm

第五章：常见问题

5.1 如果我到达机器时 jbxwriter 软件未运行怎么办？

1.
首先验证 jbxwriter 是否只是在另一个桌面上运行，检查右下角以切换所有三个桌面
2.
如果软件确实已关闭，请使用桌面上的图标打开 jbxwriter
3.
打开后，确保左下角的"开启"图标是绿色的。如果是，软件已准备好使用
4.
如果"开启"图标是蓝色的，请联系 BNC 工作人员

5.2 如何检查束流是否开启？

1.
在 jbxwriter 中，转到 Stage Control > Fixed Position，选择'FC'并点击 MOVE
2.
在 Monitor > General 下，确保电流为 0 pA (±1 pA)
3.
如果电流非零，束流已开启。关闭它：在 Image Control 下，按 Spot 按钮
4.
再次点击 Spot 关闭束流

5.3 如何执行高度映射？

1.
转到 Calibration > Material Corr. > Height Map
2.
选择适当的 Material Type / Size / Window
3.
将芯片中心（材料坐标）输入 Offset [um]
4.
定义测量阵列：Meas. Points [point] 和 Meas. Pitch [um]
5.
确保束流关闭，点击 Execute 按钮
6.
完成后点击 Log 按钮查看结果

5.4 如果看到加载错误 (ebloaderd) 该怎么办？如何避免？

重要： 此错误需要工作人员干预才能清除。如果意外遇到此错误，请立即联系 Justin 或 Bill Rowe。
避免方法：
  • 仅在 LOAD/UNLOAD 按钮亮起绿色时按下它！
  • 仅在"载物台上的晶圆盒"LED 灯熄灭时按下 EVAC/VENT 按钮！

5.5 如果 AE 或 BE 标记检测失败怎么办？

1.
转到 Calibration page → Std. Mark Detection tab
2.
在 AE Mark 子标签页，将扫描宽度从 4 改为 24
3.
在 BE Mark 子标签页，将扫描宽度从 4 改为 24
4.
点击 Execute，如果成功则点击 Save
5.
将扫描宽度改回 4，再次执行

5.6 如果 Beamer 显示"所有许可证都在使用中"错误怎么办？

如果遇到此错误并且您已预订了设备，您有权重启 PC。这将导致其他用户丢失工作，但他们应该在第一时间保存并退出 Beamer。

5.7 为什么会出现与高度测量相关的 CHIPAL 错误？

可能原因：
  • 虚拟标记的测量位置不在实际样品上
  • 位置太靠近芯片边缘或夹子
  • 由于安装不当或使用不正确的垫片环导致样品高度超出范围

5.8 为什么会出现"高度测量区域无效"的 CHIPAL 错误？

当芯片中心位置加上芯片标记位置超出晶圆盒窗口设置的"区域"时会发生这种情况。

5.9 为什么束流漂移校正执行不正确？

如果出现 DRIFT 错误，说明 SOP 未正确遵循。请重新检查：
  • 对于非对齐写入：参见 3.5.1 非对齐曝光材料校正
  • 对于对齐写入：参见 3.5.2.7 对齐束流漂移

5.10 我应该在曝光前运行 DAILYCAL 吗？

强烈建议： 不要跳过 DAILYCAL，它有助于确保获得良好结果：重新聚焦束流并最小化拼接和图案放置问题。
唯一可以跳过的情况是：
  • 您当天在同一电流下已经运行过
  • 您确定知道并信任的人在当天早些时候对您使用的电流运行过

5.11 如果有全局标记，我可以只使用虚拟芯片标记而不进行芯片标记检测吗？

不建议： 如果您关心对齐精度，应该始终同时进行全局标记检测和芯片标记检测。芯片标记检测执行更准确的重叠算法，确保每次都有很好的对齐精度。
如果您只进行全局标记检测但使用虚拟芯片标记，可能会有约 100nm 或更多的错位。
